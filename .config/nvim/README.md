# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.


1. leader-ft或C-/   以打开或隐藏终端
2. leader-/         单词来查找文件
3. leader-leader    根本文件名查找文件
4. leader-bp        固定缓冲区到最前面
5. leader-bd        关闭缓冲区


## 文本对象
- i、f   函数
    - ii  选中函数内部
    - ai  选中整个函数
- o   if或for或while
    - io    if或for内部
    - ao    外部
- c   类
- t   html标签
- d   数字
- w   单词
- e   驼峰或下划线命名的单词，只选择单词的一部分
- g   整个缓冲区
- u   函数调用
    - iu    括号内的参数，不包含括号
    - au    加上函数调用的名
